Java while loop

In computer programming, loops are used to repeat a block of code. For example, if you want to show a message 100 times, then you can use a loop. It's just a simple example; you can achieve much more with loops.

Let's explore how Java while loop program works.

Java while loop Program :-

// Program to display numbers from 1 to 5 class Main { public static void main(String[] args) { int i = 1, n = 2; while(i <= n) { System.out.println(i); i++; } } }

Output :-

1 2
